# CS480 - Introduction to Artificial Intelligence

Repository for code created in CS 480 (Spring 2016) at George Mason University.

## Quick Notes

To run the LISP command-line interface with arrow controls:

```bash
rlwrap sbcl
```

To load a .lisp file:

```lisp
(load myfile.lisp)
```
